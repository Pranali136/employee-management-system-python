# Employee Management System

A console-based Employee Management System developed using Python and Object-Oriented Programming (OOP) concepts. This project allows users to manage employee records through a menu-driven interface while demonstrating key OOP principles such as inheritance, encapsulation, and method overriding.

## Features

* Add Employee
* Add Developer
* Display Employees
* Search Employee by ID
* Delete Employee
* Employee Email Generation
* Salary Management
* Menu-Driven Interface

## Technologies Used

* Python
* Object-Oriented Programming (OOP)

## OOP Concepts Implemented

* Classes and Objects
* Inheritance
* Encapsulation
* Method Overriding
* Constructors
* Modular Programming

## How to Run

1. Clone the repository

git clone https://github.com/Pranali136/employee-management-system-python.git

2. Navigate to the project folder

cd employee-management-system-python

3. Run the application

python main.py
